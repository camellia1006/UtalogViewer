# Utalog Viewer

## 概要
- スプレッドシート経由でデータを読み込み、テーブルで一覧表示します。
- テーブルの行を選択すると、割り当てられた動画が再生されます。

## 機能

- **動画再生**
  - 選択された動画を再生します。
  - プラットフォーム側の制限などで再生できない場合には、元サイトへのリンクから視聴できる場合があります。

- **フィルター**
  - [検索]ボックスに入力された内容に部分一致する行のみを表示します。

- **ソート**
  - テーブルの列名を選択すると、その列をキーにして昇順/降順に並べ替えをします。
  - 同じ値だった場合には直前のソート順が適用されます。

- **カスタマイズ**
  - URLパラメータで値を指定することで、機能やレイアウトを変更できます。
  - 必須
    - id / name
      - 読み込むデータを指定します。
      - 無効な値は無視されます。
      - 両方指定した場合は、idが優先されます。
  - 任意
    - autoplay (既定値: 0)
      - 動画の自動再生を変更します。
      - ブラウザの制限により機能しない場合があります。
    - wrap (既定値: 1)
      - テーブル内の改行を変更します。
      - 改行しない場合、縦のスクロールが見やすくなりますが、横に大きく広がります。
