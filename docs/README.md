# Utalog Viewer

## 概要
スプレッドシート経由でデータを読み込み、テーブルで一覧表示する。
テーブルの行を選択すると、割り当てられた動画が再生される。

## 機能

- **動画再生**
  - 選択された動画を再生する。
  - 不具合やプラットフォーム側の制限などで再生できない場合には、元サイトへのリンクから視聴できる場合があります。

- **フィルター**
  - [検索]ボックスに入力された内容に部分一致する行のみを表示する。

- **ソート**
  - テーブルの列名を選択すると、その列をキーにして昇順/降順に並べ替えをする。
  - 同じ値だった場合には直前のソート順が適用される。
